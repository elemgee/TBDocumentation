excelCellTypeForProperty (string)

D2W only

this set the cell type for an excel export.

also check excelCellWidth, excelCellFormatForProperty,
excelDefaultColumnWidthForSheet, excelFileName and displayPropertyKeys.

Sample :
	100 : (PC = 'ListExcel#' and PK = 'attribute1') => excelCellTypeForProperty = "CELL_TYPE_NUMERIC" [com.webobjects.directtoweb.Assignment]
