DPK

DPK [D3W only], called in D2W 'displayPropertyKeys'

Format : the Format is an plist Array '("[TabName]", "(SectionName)", "Attribute" ...)'.

only the List view has no Tab and Section.

Localization :

    * Tabs can be localized in the Localization.strings file
        "TabKeys.name" = "localizedName";
    
    * TODO ... more coming

Combined Properties :

    * Writing whitespacesget trimmed out
    
    * combine 2 Attributes with the '|' character' let display in 1 column
    
        "attributeName1 | attributeName2"
        
    * TODO ... more coming

OptionalTag :

    * in Edit mode you can use the '@static' keyword that makes it readonly

        "firstName@static"
     
	* starting a property with '-' will show the propertyName but will automatically
		hide the value and instead show a hr tag. so good for debugging to remove a
		property quickly.
		
		"-firstName"
     
	* you can use 'dummy' as propertyName, this will hide the PropertyName and for
		the value it will display a markdown file from the Resource folder.     
     
     	"dummy@option=WOdka:RoleHelper.md"
     	
     	'WOdka:RoleHelper.md' = [[FrameworkName]]:[[FileName]]
     
     
    * TODO ... more coming
