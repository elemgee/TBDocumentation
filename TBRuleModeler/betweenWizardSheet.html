betweenWizardSheet (string)

you have to set your ComponentName for using this Feature.
It will add your Component between current Page with the Add button and
before it goes to the real Wizardpage Component.

Sample :
	620 : *true* => betweenWizardSheet = "ComponentBetweenWizardSheet" [com.webobjects.directtoweb.Assignment]:

D3W Usage : only 
D2W Usage : n/a
