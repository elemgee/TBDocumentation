defaultBatchSize (number)

This rule will set the default batch size for a list page, and decide how many Objects will be in the repetition. If it is set to "0" then all objects will be returned in the repetition.


700 : PC = 'ListEmbedded#' => defaultBatchSize = "7" [com.webobjects.directtoweb.Assignment]

700 : PC = 'List#' => defaultBatchSize = "0" [com.webobjects.directtoweb.Assignment]
