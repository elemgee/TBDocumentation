importDelegateName (string)

this will set the classPath which Importer Delegate Object has to be Execute.

Sample :
	720 : EN = '#' => importDelegateName = "com.ksroom.delegate.csv.PersonImporter" [com.webobjects.directtoweb.Assignment]

D3W Usage : in ButtonDelegate using CSV importer
D2W Usage : NO (D3W only)
