excelCellWidth (number)

D2W only

this set the cell width for an excel export.

also check excelCellTypeForProperty, excelCellFormatForProperty,
excelDefaultColumnWidthForSheet, excelFileName and displayPropertyKeys.

Sample :
	100 : (PC = 'ListExcel#' and PK = 'attribute1') => excelCellWidth = "10" [com.webobjects.directtoweb.Assignment]
