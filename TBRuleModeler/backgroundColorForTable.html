backgroundColorForTable

Used throughout the original D2W Templates. Determines the color for various uses of tables and navigation bars.

If the alternateRowColor key is set to true for a list page, then the value for 'backgroundColorForTable' is used for one set of rows and the automatically generated darker color is used for the alternating rows.

From D2WContextKeys document:

Type: String

Appears: Rule's key (right-hand side)

The background color for the table on page containing the receiver.

Example usage

50 : *true* => backgroundColorForTable = "#ebebeb" [com.webobjects.directtoweb.Assignment]
