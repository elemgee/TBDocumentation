batchSize

Used in ERD2WListPage. The value can be overridden by user preferences if they have been enabled.

From D2WContextKeys document:

Type: Integer

Appears: Rule's key (right-hand side)

The number of objects per batch displayed on a list page. This key is used by the list and plain-list page Direct to Web templates.

Example usage

200 : task = 'select' => batchSize = "10" [com.webobjects.directtoweb.Assignment]

600 : targetOutput = 'xml' => batchSize = "0" [com.webobjects.directtoweb.Assignment]
