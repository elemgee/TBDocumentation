yearRangeTop (number)

specifies the upper limit for year to be displayed

Sample :
	100 : *true* => yearRangeTop = "2030" [com.webobjects.directtoweb.Assignment]

D3W Usage : 
	D3W_Edit_dateAsPopup (default "2050")

D2W Usage : 
	ERDEditDatePopup (default "2050")
	
@see :
	yearRangeBottom
