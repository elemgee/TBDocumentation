actions

This key is used to define which action buttons will appear in a D2W page. See also "isEntityInspectable", "isEntityEditable", "readOnly", "isEntityDeletable" their short forms "crud" and "cruds", and "isEntityPrintable"

Actions that are defined in the D2W rule system are "inspectAction", "editAction", "deleteAction", "removeRelatedAction", "selectAction", "controllerAction", "printAction"

Positions that are defined for buttons are "left", "right", "center", "top", and "bottom"

Example usage

100 : task = 'select' => actions = {"left" = ("selectAction"); } [com.webobjects.directtoweb.Assignment]

100 : pageConfiguration = 'ListDocument' => actions = {"right" = ("editAction", "controllerAction"); } [com.webobjects.directtoweb.Assignment]

100 : pageConfiguration = 'ListDocument' => actions = {"" = ""; } [com.webobjects.directtoweb.Assignment]

20 : *true* => actions = {"left" = ("controllerAction"); "right" = ("deleteAction"); } [com.webobjects.directtoweb.Assignment]

Note that 'pageConfiguration' can be shortened to 'PC'. Thus the rules can also be written as follows:

100 : task = 'select' => actions = {"left" = ("selectAction"); } [com.webobjects.directtoweb.Assignment]

100 : PC = 'ListDocument' => actions = {"right" = ("editAction", "controllerAction"); } [com.webobjects.directtoweb.Assignment]

100 : PC = 'ListDocument' => actions = {"" = ""; } [com.webobjects.directtoweb.Assignment]

20 : *true* => actions = {"left" = ("controllerAction"); "right" = ("deleteAction"); } [com.webobjects.directtoweb.Assignment]
