excelDefaultColumnWidthForSheet (number)

this set the default cell width for an excel export.

also check excelCellWidth, excelCellTypeForProperty, excelCellFormatForProperty,
excelFileName and displayPropertyKeys.

Sample :
	100 : PC = 'ListExcel#' => excelDefaultColumnWidthForSheet = "40" [com.webobjects.directtoweb.Assignment]
