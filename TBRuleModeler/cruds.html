cruds (string)

only D3W

(C)reate, (R)ead, (U)update, (D)elete, (S)earch on this EO for User Interface buttons.

This uses a "byte" pattern. Each ordinal position corresponds with the action of Create, Read, Update, Delete.

0 == Action NOT Allowed. 1 == Action Allowed.

10000 == display the Add button.

01000 == display the Inspect on an EO button.

00100 == display the Edit button.

00010 == display the Remove/Delete button.

00001 == display the Find/Query button.
