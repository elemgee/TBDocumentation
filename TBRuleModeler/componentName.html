componentName

From D2WContextKeys document:

Type: String

Appears: Rule's key (right-hand side)

The name of a property-level component that can be used to display a particular property for a particular task and entity. This key is used by the Direct to Web templates.

Example usage

10 : propertyKey = 'deleteAction' => componentName = "ERDDeleteButton" [com.webobjects.directtoweb.Assignment]

25 : propertyKey = 'url' => componentName = "ERD2WDisplayURL" [com.webobjects.directtoweb.Assignment]
