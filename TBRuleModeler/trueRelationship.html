trueRelationship (string)

This key is used with d2wType 'toManyIsTrue'.

In display mode it will display the releationship with the boolean key and value is true.
false value will not displayed.
In edit mode it will display all relationships with a checkbox to select on/off

Example usage

600 : (EN = '#' and PK = '{propertyName}') => trueRelationship = "{booleanAttributeName}" [com.webobjects.directtoweb.Assignment]


you can also write dirctly "{propertyName}@option={booleanAttributeName}" into DPK.
