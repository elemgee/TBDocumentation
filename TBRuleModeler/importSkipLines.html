importSkipLines (number)

this will set how many lines in the csv is a header and will not be used.

Sample :
	720 : EN = '#' => importSkipLines = "1" [com.webobjects.directtoweb.Assignment]

D3W Usage : in ButtonDelegate using CSV importer
D2W Usage : NO (D3W only)
	
@see :
	importPropertyKeys
	importDelegateName
