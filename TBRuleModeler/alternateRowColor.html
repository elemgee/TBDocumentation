alternateRowColor

This boolean key is used to tell a list page whether to alternate row colours in the table or not. If the answer is 'true' then the backgroundColorForTable key is used for one row and an automatically generated 'darker' color is used for the alternate row.

From D2WContextKeys document:

Type: Integer

Appears: Rule's key (right-hand side)

A flag that has a value of 1 if the list page displays its entries with alternating row colors and a value of zero otherwise. This flag is used by some list page Direct to Web templates.

Example usage

0 : *true* => alternateRowColor = "false" [com.webobjects.directtoweb.BooleanAssignment],
