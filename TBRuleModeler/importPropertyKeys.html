importPropertyKeys (Array String)

this is like a DPK but without Tab's and Section's and it define keywords that you will get in your Importer.

Sample :
	720 : EN = '#' => importPropertyKeys = ("uid", "loginName", "firstName", "lastName", "primaryEmailAddress", "loginAllowed", "domain", "role") [com.webobjects.directtoweb.Assignment]

D3W Usage : in ButtonDelegate using CSV importer
D2W Usage : NO (D3W only)
	
@see :
	importDelegateName
	importSkipLines
