cmsLanguageSwitching (boolean)

If this key is set to true the plist of the cms url for the header and footer will depend on the language, so it will possible to create other content for each language.

Example usage

700 : EN = '#' => cmsLanguageSwitching = "false" [com.webobjects.directtoweb.BooleanAssignment]
