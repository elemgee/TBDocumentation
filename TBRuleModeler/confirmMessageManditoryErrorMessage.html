confirmMessageManditoryErrorMessage (string)

This is used in the Confirmpage.

this display the errormessage if the confirm is Manditory.

default Message is : You must enter a confirmation message.

Sample :
	100 : *true* => confirmMessageManditoryErrorMessage = "You must enter a confirmation message" [com.webobjects.directtoweb.Assignment]
