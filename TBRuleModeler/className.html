className

Used to associate the ENUM attribute with the Java ENUM class. This is used in pair with an "enum" or "enumAsCheckboxes" keys.

	Sample:
	(entity.name = 'Xxx' and propertyKey = 'xxx') => className = "me.webobjects.enums.WOdkaPaymentsEnums" [com.webobjects.directtoweb.Assignment]
