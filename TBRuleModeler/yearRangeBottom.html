yearRangeBottom (number)

specifies the lower limit for year to be displayed

Sample :
	100 : *true* => yearRangeBottom = "2001" [com.webobjects.directtoweb.Assignment].

D3W Usage : 
	D3W_Edit_dateAsPopup (default "1950")

D2W Usage : 
	ERDEditDatePopup (default "1950")
	
@see :
	yearRangeTop
