d2wType (string)

only D3W

D2WType is an attribute right-hand side key. Use this to define the UI component to use for this attribute defined on the left-side.

The template engine will generate default inactive rules with D2WType defined for each attribute. You should activate these rules and change the type if needed.

If you have a LONGTEXT you will need to change the D2WType from "string" to "tiny".

Some commonly used right side values>/p>

"amount" for currency attributes.

"attachment" for ERAttachment attributes.

"enum" for enum attributes. See "classname" right-hand side key.

"enumsAsCheckboxes" for enum attributes. See "classname" or "showImage" right-hand side key.
